Problem Description:

This project focuses on Lexical Sample WSD for Urdu language, the purpose of the project 
is to disambiguate pre defined ambiguous words appear in the text automatically.

-The Data folder contains the CSV file of the corpus (UAW-WSD-18 Corpus) used for experementation (various methods of feature engineering).   
-The excel fle contains complete experemental results
-Python files contans the complete code, which we developed for Lexical Sample WSD using Keras and Tenserflow (Lexical_Sample_Automatic_Features contains the code of Automatic extracted features, where Lexical_Sample_Manual_Features contains the code of manually extracted features).
